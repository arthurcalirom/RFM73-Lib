Value sent to first argument of rfm73_receive_packet function. In this case function will automatically transmit received message back.

#define RFM73_RX_WITH_NOACK   0

#define RFM73_TX_WITH_ACK   1

Value sent to first argument of rfm73_send_packet function. Enables auto-acknowledge of sending packet. Auto-acknowledge must be explicitly enabled by rfm73_set_autoack function.
