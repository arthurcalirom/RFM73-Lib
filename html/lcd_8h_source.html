57  * void LCD4_Write_tetrad(unsigned char data)
58  Подпрограмма записи одной из тетрад байта в ЖКИ.
59  Производит сдвиг на LCD4_D_PIN_SHIFT всех разрядов
60  переменной data, поскольку биты данных могут располагаться

62  data на порте и дергает ножку LCD4_E_PIN

69  3. Специализированные
70 
71  * void lcd_set_cgram(char adr, char data)
72  Запись в CGRAM байта data по адресу adr. adr=0..3FF, data=0..1F
73  Всего можно адресовать 8 символов, которые будут иметь порядковые
74  номера 0..7. старшие 3 разряда adr определяют порядковый номер
75  символа adr, младшие - порядковый номер строки в символе. Возможен
76  только режим 5х7, так что можно нарисовать только символ 5х8,
77  поскольку еще 1 строчка есть под курсор.
78  Единица в поле data означает, что соответствующий пиксель
79  соответствующей строки соответствующего символа будет гореть.
80  Низкоуровневая функция, следует использовать lcd_set_cg_symbol
81 
82  * void lcd_set_cg_symbol(char adr, flash char s[8])
83  Перезапись символа в CGRAM по адресу adr. 8 раз вызывает функцию
84  lcd_set_cgram. Второй аргумент - массив, записываемый в CGRAM,
85  хранящийся во flash (при желании можно заменить на eeprom,
86  переписав соответствующее поле).
